## Production Workflow

### Pre-Production

- [ ] Write detailed script
- [ ] Create visual assets (logos, screenshots)
- [ ] Set up demo environment with realistic data
- [ ] Test recording setup

### Production

- [ ] Record voiceover separately (better audio quality)
- [ ] Record screen captures
- [ ] Record talking head intro/outro (optional)

### Post-Production

- [ ] Edit in DaVinci Resolve or Premiere Pro
- [ ] Add:
  - [ ] Music
  - [ ] Sound effects (subtle)
  - [ ] Lower thirds (text overlays)
  - [ ] Captions
  - [ ] End screen (CTA)
- [ ] Color grading for brand consistency
- [ ] Export at 1080p, 60fps

### Publishing

- [ ] Upload to YouTube
- [ ] Create custom thumbnail (1280x720)
- [ ] Optimize title and description for SEO
- [ ] Add timestamps in description
- [ ] Share on social media

### Professional Setup

- Microphone: Shure SM7B ($400)
- Audio interface: Focusrite Scarlett ($150)
- Camera: Sony A6400 ($900) for talking head
- Lighting: Key + fill + backlight ($200)
